  - ordered_list_3
스타일 매핑:
  "1. " (1단계):        번호1
  "   1. " (2단계):     번호2
  "      1. " (3단계):  번호3
테스트 관점:
  - 원문의 번호 텍스트를 자동 재번호매김 없이 문자 그대로 보존하는가
    예를 들어 "1. / 1. / 1." 로 입력되면 세 항목 모두 "1." 로 출력됨
    "1. / 2. / 3." 로 입력되면 "1. / 2. / 3." 그대로 출력됨
  - 1 단계와 2 단계 사이 들여쓰기 3 칸, 2 단계와 3 단계 사이 추가 3 칸 규칙이 인식되는가
  - 번호1 / 번호2 / 번호3 은 글머리 기호 없는 들여쓰기 단락 스타일이므로 번호 텍스트 자체는 문단 본문에 포함됨
---
