마크다운의 image(![alt](url) 문법) 에 대응하는 hwpx 스타일을 지정함. hwpx 에는 그림을 글자로 취급할 경우 글자에 해당하는 스타일을 지정할 수 있음. "그림"은 이에 해당하는 스타일임. 이 스타일은 이미지가 삽입된 단락에 적용되며 가운데 정렬을 기본으로 함. 

그림 제목에 해당하는 단락은 hwpx 에서 캡션 설정으로 추가되는 캡션 문단에 "그림캡션" 스타일로 부여함. 그림의 경우 캡션의 위치는 표와 달리 그림 아래임. 마크다운에서 그림 직후 단락이 "그림 1." 또는 "Figure 1." 로 시작하는 경우 변환기가 해당 단락을 캡션으로 승격하여 그림 바로 아래에 배치함. 



아래는 ast_walker.py 에서 그림 캡션 승격 로직을 처리하는 부분의 예시임.
import re

# 그림 캡션 감지 정규식
# "그림 1.", "그림 2.", "Figure 1.", "Figure 12." 등에 매칭
그림캡션패턴 = re.compile(r'^(그림|Figure)\s+\d+\.\s*')

def 그림캡션감지(단락):
    """단락 텍스트가 그림 캡션 형식으로 시작하는지 판정."""
    if 단락.역할 != "paragraph":
        return False
    return bool(그림캡션패턴.match(단락.텍스트.strip()))

    """AST 블록 목록을 순회하며 그림 직후 단락이 캡션 패턴을
    가지면 "figure_caption" 역할로 승격함."""
    결과 = []
    i = 0
    while i < len(블록목록):
