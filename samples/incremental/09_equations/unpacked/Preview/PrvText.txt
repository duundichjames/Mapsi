마크다운의 수식 문법($...$ 인라인과 $$...$$ 디스플레이) 에 대응하는 hwpx 요소를 생성함. hwpx 는 수식을 hp:equation 요소로 표현하며 내부 hp:script 에 한/글 자체의 수식 언어(HNC 수식) 로 작성된 문자열을 담음. HNC 수식은 LaTeX 와 문법이 유사하지만 세부 명령어와 구조가 다르므로 변환기가 LaTeX 를 HNC 수식으로 변환해야 함. 예를 들어 LaTeX 의 \frac{a}{b} 는 HNC 에서 {a} over {b} 로, \sum_{i=1}^{n} 는 sum_{i=1} ^{n} 으로, \begin{matrix}...\end{matrix} 는 matrix{... # ...} 로 표현됨.

이 변환은 결정론적 규칙 엔진으로 완성하려면 LaTeX 전체 문법을 파싱해야 하므로 범위가 커 2일 스프린트 내에 구현하기 어려움. 대신 변환기는 LLM API 의 사용 가능 여부에 따라 두 경로를 선택함. 환경 변수에 OpenAI 또는 Anthropic 의 API 키가 설정되어 있으면 LLM 을 호출하여 LaTeX 를 HNC 수식으로 변환하고, 그 결과를 hp:script 에 삽입함. API 키가 없으면 LaTeX 원문을 그대로 보존하되 앞에 [hnc 수식] 태그를 붙여 사용자가 변환 결과에서 해당 위치를 찾아 한/글 수식 편집기로 직접 입력할 수 있도록 안내함.

LLM 호출 시 한/글이 공개한 수식 스펙 문서(한글 문서 파일 구조 - 수식) 를 컨텍스트로 함께 전달하여 변환 정확도를 높임. 이 문서는 HNC 수식의 명령어, 기호 이름, 문법 규칙을 정리하고 있으며 LLM 에게 변환 규칙의 근거 자료로 활용됨. 반복 호출을 줄이기 위해 변환 결과를 로컬 JSON 캐시에 (LaTeX 원문 → HNC 수식) 쌍으로 저장하며, 같은 LaTeX 가 다시 등장하면 캐시에서 바로 반환함.
